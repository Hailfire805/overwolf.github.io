* Manifest flag [enable_top_isolation](manifest#enable_top_isolation) has been "moved up" in the hirarchy:  
  Now it's per app and not per windows - means, it's under the "data" node and not the "window" node.  
  Note that it's backward compatible (means that if one of the app windows set this flag in the manifest - it will auto-merge to the entire app).
* New function: [overwolf.utils.OpenFilePicker](utils#openfilepickerfilter-initialpath-callback).  
  Now also accepts an initialPath parameter.
* Logs: now end with the extension "log" (instead of numbers).  
  In addition, logs file names are now counted where the last number one is the newest file.  
* New function: [overwolf.streaming.split](streaming#splitstreamid-callback).  
  Split video files.
* Implement the main features of the [IO-Pluign](/topics/using-plugins/simple-io-plugin) inside the [overwolf.io](io) API: 
  * [overwolf.io.dir()](io#dirpath-callback)
  * [overwolf.io.readBinaryFile()](io#readbinaryfilepath-options-callback)
  * [overwolf.io.readTextFile()](io#readtextfilepath-options-callback)
  * [overwolf.io.exist()](io#existpath-callback)
  * [overwolf.io.listenOnFile()](io#listenonfileid-path-option-callback)
  * [overwolf.io.stopFileListener()](io#stopfilelistenerid)
