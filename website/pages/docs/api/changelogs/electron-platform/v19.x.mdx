## API

- New API - [`app.overwolf.disableAnonymousAnalytics()`](/tools/ow-electron#anonymous-analytics):
  - Allows an app to disable the sending of anonimized app analytics to Overwolf from the moment it is ran.
- New API - [`app.overwolf.isCMPRequired()`](/api/electron/consent-management-platform):
  - Checks whether or not the local user must accept the [CMP](/tools/ow-electron/consent-management-platform) in order to use the app.
- New API - [`app.overwolf.openCMPWindow()`](/api/electron/consent-management-platform):
  - Opens the [CMP settings window](/tools/ow-electron/consent-management-platform) for the user, allowing them to customize their consent settings.

## Platform

- Overwolf APIs are now mapped in the `electron.d.ts` file.
