#### *mob_details* note

List of parameters:

* name - creature name
* grade - creature grade
* health - current creature health
* health_max - max health of the creature
* level - creature level

Data Example:

```json
{"info":{"match_info":{"mob_details":"{\"name\":\"Large Rock Boar\",\"grade\":\"NORMAL\",\"health\":413,\"health_max\":3192,\"level\":31}"}},"feature":"match_info"}
```

#### *match_start* note
