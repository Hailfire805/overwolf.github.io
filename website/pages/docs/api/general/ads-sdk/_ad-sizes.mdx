The following Ad container sizes are currently supported by the Ads SDK:

| Container size            | Supported display Ads                 | Supported video Ads         | (OW) Size value snippet                       |
| ------------------------- | ------------------------------------- | --------------------------- | --------------------------------------------- |
| 400x300                   | 336x280 / 300x250 / 250x250           | 400x300 / 300x250           | `{ width: 400, height: 300 }`                 |
| 400x600                   | 336x280 / 300x600 / 300x250 / 250x250 | 400x300 / 300x250           | `[{ width: 400, height: 600 }, { width: 400, height: 300 }]` |
| 300x250                   | 300x250 / 250x250                     | N/A                         | `{ width: 300, height: 250 }`                 |
| 160x600                   | 160x600 / 120x600                     | N/A                         | `{ width: 160, height: 600 }`                 |
| 728x90                    | 728x90                                | N/A                         | `{ width: 728, height: 90 }`                  |

:::warning Ad Sizes Are Precise!
Make sure to only enter values from the `Size value snippet`! Any unsupported values will potentially cause issues!

For specific edge-cases, or if you wish to use several different sizes in a single container, [contact us](/support/contact-us).
:::
